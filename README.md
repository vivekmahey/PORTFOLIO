## 💡 About the Project

This portfolio represents my journey as a developer and creator. It combines simplicity with interactivity, featuring an elegant hero section with a 3D Spline robot model and smooth, modern UI transitions.  
The site was built using clean HTML, TailwindCSS, and JavaScript, designed for both performance and aesthetics.

- ⚡ **Responsive Design** – Works seamlessly across all devices.  
- 🎨 **Modern UI** – Clean layout with glowing highlights and smooth animations.

## 🧑‍💻 Setup & Installation

If you’d like to run this project locally, follow these steps:

1. Clone this repository:
   ```bash
   git clone https://github.com/vivekmahey/PORTFOLIO.git
````

2. Open the project folder:

   ```bash
   cd PORTFOLIO
   ```
3. Open the main file:

   ```bash
   index.html
   ```
4. Open it in your browser and explore the portfolio locally.

---

## 🌍 Deployment (GitHub Pages)

To deploy your portfolio live:

1. Go to your repository settings on GitHub.
2. Scroll down to **Pages**.
3. Under **Source**, select **main branch** → `/ (root)`.
4. Click **Save**.
5. Your site will be live at:

   ```
   https://vivekmahey.github.io/PORTFOLIO/
   ```
